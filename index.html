Unit Step(Heaviside)

Basically, the Heaviside step function, or the unit step function, usually denoted by a slanted H or θ (but sometimes u, 1 or 𝟙), is a discontinuous piece wise function where the output is 0 for negative inputs, 0.5 for an input of zero, and 1 for positive inputs. This function is like a more rough approximation of the logistic function. The unit step function is most often utilized as a distribution; and, is how the function's used in neural networks. Moreover, in neural networks, the function tries to play the role of a neuron by creating an artificial threshold where the neuron will spike. These artificial neurons can be found in the multiperceptron model. Unfortunately, the unit step function loses a lot of important information --for the back propagation and loss functions-- in the process of firing.


Sign (Signum)

The sign function(or signum function) is basically a variation of the step function. So, the function is used in neural networks identically and has a distribution that looks extremely similiar. However, the sign function introduces negative values to activation functions and is an odd mathematical function.


Linear

The linear activation function is pretty much useless because it takes away the idea of deep learning all together. So, you really shouldn't pay much attention to this type of activation function.


Piece-wise Linear

The piece-wise linear function, unlike the linear, is a nonlinear activation function, so it can be used to train neural networks. Basically, the idea of this one is to introduce nonlinearity with the least amount of work possible. However, the minimilistic model of the piece-wise linear function does not make it the best activation function for every situation.


Logistic (sigmoid)

The logistic function is used a lot due to it's smooth nature. So, the function can map any output to a probability between 0 and 1: allowing the loss function to be more useful.


Hyperbolic tangent

Hyperbolic tangent function, or tanh function, is a variation of the logistic function. The function is different in the way that it can have a negatve output, but everything else is pretty much the same.
